Quick Start

Up and running
in minutes.

Choose your integration method and start deploying AI specialists immediately.

1
Claude Code (Recommended)
Copy agents directly into your Claude Code directory and activate any specialist instantly.
# Copy agents to Claude Code
cp -r agency-agents/* \
  ~/.claude/agents/

# Activate in your session:
# "Use the Frontend Developer
#  agent to build a component"
2
Multi-Tool Install
Convert and install agents for Cursor, Aider, Windsurf, Gemini CLI, or OpenCode in one step.
# Generate integration files
./scripts/convert.sh

# Interactive install
# (auto-detects your tools)
./scripts/install.sh

# Or target a specific tool
./scripts/install.sh \
  --tool cursor
3
Activate Your Specialists
Reference any agent in your AI conversations and watch the specialists take over.
# Claude Code examples:

"Use the Frontend Developer
 agent to review this React
 component."

"Activate the Reality Checker
 to verify production
 readiness."

"Use the Growth Hacker to
 plan our launch strategy."
4
Orchestrate a Full Team
Deploy multiple specialists simultaneously for complex projects needing cross-functional expertise.
# Full agency deployment:

"Use the Agents Orchestrator
 to coordinate Backend Arch,
 Frontend Dev, Growth Hacker,
 and Reality Checker for
 our MVP launch."
